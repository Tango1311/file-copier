# Копировщик файлов в Git репозиторий (Вариант 9)

## Описание
Графическое приложение для копирования файлов в локальный Git репозиторий.
Скрипт позволяет выбирать файлы и копировать их в указанную папку репозитория.

## Функциональность
- Выбор исходного файла через графический интерфейс
- Выбор папки локального Git репозитория
- Создание папки назначения внутри репозитория
- Проверка наличия Git репозитория
- История копирований
- Обработка ошибок

## Использование
1. Запустите скрипт: `python file_copier.py`
2. Выберите исходный файл
3. Выберите папку локального репозитория
4. Укажите папку назначения
5. Нажмите "Копировать файл в репозиторий"

## Требования
- Python 3.6+
- Git (установленный в системе)
